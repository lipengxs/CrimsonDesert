# Crimson Desert

Welcome to the official website repository for Crimson Desert, an epic open-world action adventure game developed by Pearl Abyss.

## Overview

This repository contains the website code for Crimson Desert, featuring:
- Latest news and updates
- Media gallery with screenshots and videos
- Game guides and strategy tips
- Legal documentation

## Game Information

**Crimson Desert** is an open-world action adventure game scheduled for release on **March 19, 2026**. The game follows the story of Kliff and the Greymanes mercenary group as they explore the vast continent of Pywel. Players will experience intense combat, deep exploration, and immersive storytelling in a beautifully crafted open world.

## Structure

The website includes the following main sections:
- `/` - Home page with game introduction, features, and countdown timer
- `/news` - Latest updates, announcements, and events
- `/media` - Screenshots, gameplay videos, and artwork gallery
- `/guides` - Comprehensive game guides covering combat, exploration, and strategies
- `/privacy-policy` - Privacy policy documentation
- `/terms-of-service` - Terms of service documentation

## Features

- **Multi-language Support**: English, Chinese (Simplified), and Japanese versions
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Countdown timer, image lightbox, FAQ accordion, and more
- **SEO Optimized**: Comprehensive meta tags and structured content

## Development

### Prerequisites
- Modern web browser
- Web server (for local development)

### Setup
1. Clone the repository
2. Open `index.html` in a web browser or use a local web server
3. For development, you can use Python's built-in server: `python -m http.server 8000`
